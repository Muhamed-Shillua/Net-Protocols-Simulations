Network Protocol Simulation

A simple demonstration using C++ and raw sockets

Purpose of the Simulation

This project simulates how different networking protocols (like TCP, UDP, HTTP) work at a basic level using C++.

What You Can Learn

  • How sockets are used in C++ for communication
  • The difference between TCP and UDP
  • How HTTP responses are sent from a server to a browser

How It Works

The server listens on a specific port and responds with a static HTML file when a client (like your browser) connects.

Example

Run the server, then open http://localhost:9090 in your browser to see this page served by the C++ server.
